Welcome to the About

Introduction

This project deals with the storage and display of the data analytics of the Rose-Hulman Greenhouse.

Overview

There are many sensors and actuators that are located within the Greenhouse that are constantly obtaining and transmitting information about the Greenhouse (i.e. Temperature, Humdity, etc). This website was created in order to allow for any student or staff from Rose-Hulman to be able to access old sensor readings as well as view current sensor readings. This website also has a graphic display which allows individuals to be able to see trends in the sensors readings.

How-To-Use

If the Home button is selected, you will be transfered to the homepage of the website where images of the Greenhouse will be shown.
If the Grab Current Data button is selected, you will be able to see the current sensor readings of the Greenhouse, as well as be able to see a graphical representation of the trends of the data.
If the View Histrorical Data button is selected, you will be able to view all the sensor readings that are stored on the cloud.
If the About button is selected, you will stay on this page.
